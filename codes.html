LIM simulator

This tool simulates galaxy line-intensity maps from simulated halo catalogs. It does so by employing scaling relations to relate halo mass to line-luminosities ([C II]158\(\mu\)m and CO) and gridding them over a cosmological volume. This package is entirely written in C++ and is avaialable here: https://github.com/chandra-001/LIM_simulator

Power Spectrum

This tool can estimate the spherically averaged 3D power spectrum of a given line-intensity mapping signal over a cosmological volume. It can be used on any cosmological field as well. Written entirely in C++, it is available here: https://github.com/chandra-001/PowerSpectrum
